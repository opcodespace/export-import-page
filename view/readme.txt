With this plugin you simply export and import page, post and custom post. This plugin supports ACF fields.

== Description ==
This "Simple Export Import for ACF Data" plugin is very helpful for developers who develop theme locally with ACF and deploy to live.

= Supported ACF fields in FREE Plugin =
1. Basic
* Text
* Text Area
2. Cointent
3. Choice
4. Relational
5. jQuery
6. Layout

= Supported ACF fields in Paid Plugin =

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the `Settings->Simple Export Import` screen to export & Import data

== Frequently Asked Questions ==

= Does it support fields of `Advanced Custom Field` plugin? =
Yes, free plugin is supporting repeater, groups and flexible layouts. For image export, import you need pro plugin.

= Can I export and import as bulk? =
Yes, you can but you need pro plugin.

= Can I export and import options data? =
Yes, you can but you need pro plugin.
